# 舆情监测系统

## 项目说明
这是一个基于Python的微博内容词云生成器，可以从CSV文件中读取微博内容，并生成美观的词云图。

## 功能特点
- 支持微博内容和视频描述的处理
- 智能文本清理（表情符号、URL、@用户等）
- 中文分词与停用词过滤
- 自定义背景图支持
- 智能颜色适配

## 依赖要求
```
jieba
wordcloud
numpy
pandas
Pillow
```

## 使用方法
1. 准备CSV文件（包含'微博内容'或'视频描述'列）
2. 准备背景图片（命名为'xuexi.jpg'）
3. 准备字体文件（'三极泼墨体.ttf'）
4. 运行程序：
```python
python model/ciyuntu.py
```

## 团队成员头像配置
团队成员头像需要放置在 `static/assets/images/team/` 目录下，命名规则如下：
- 文梓婧.jpg
- 彭静怡.jpg
- 罗相吉.jpg
- 赵雯祺.jpg
- 陈旭豪.jpg
- 陈俊松.jpg

请确保图片为jpg格式，建议尺寸比例为4:3（宽:高），推荐分辨率为160x120像素。
