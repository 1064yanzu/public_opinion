# ClarityStudio 登陆页面

这是一个现代化、美观的 ClarityStudio 产品登陆页面，用于展示产品功能和吸引用户使用。

## 项目特点

- 现代化、高级的设计审美
- 响应式布局，适配各种设备尺寸
- 使用纯 HTML, CSS 和 JavaScript 开发
- 包含动画和交互效果
- 字体和资源确保在中国大陆可以正常加载

## 文件结构

- `index.html` - 主页HTML文件
- `styles.css` - 样式表文件
- `script.js` - JavaScript交互脚本
- `images/` - 图片资源目录
  - `logo.svg` - 网站Logo

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可预览页面
2. 可以部署到任何支持静态网站托管的服务上

## 主要功能区

1. 导航栏 - 包含链接和登录/注册按钮
2. 英雄区 - 展示产品主要价值
3. 功能区 - 介绍产品主要功能
4. 工作原理 - 解释产品如何工作
5. 价格方案 - 展示产品价格（当前为限时免费）
6. 页脚 - 包含版权信息

## 自定义

可以通过修改 CSS 变量轻松更改网站主题色、字体和其他设计元素：

```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #ffffff;
    --accent-color: #3d5af1;
    // 其他变量...
}
```

## 注意事项

- 确保所有资源文件的路径正确
- 使用支持现代CSS和JavaScript特性的浏览器查看最佳效果
